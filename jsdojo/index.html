// wild js facts

    

      

        
🔒 Primitives

        
Primitives are immutable. let x = "hello"; x[0] = "H" silently does nothing. The string stays "hello".

      

      

        
🤯 typeof null

        
typeof null === "object" — this is a 30-year-old bug in JS that was never fixed because too much code depends on it.

      

      

        
⚡ Born in 10 days

        
Brendan Eich wrote the first version of JavaScript in just 10 days in 1995. The quirks make more sense now.

      

      

        
🎭 Truthy magic

        
true && "hello" returns "hello". false && "hello" returns false. The && operator returns values, not just booleans.

      

      

        
🔄 Type coercion

        
"5" - 3 gives 2 but "5" + 3 gives "53". Subtraction forces numbers; addition sees a string and concatenates.

      

      

        
📦 NaN is a number

        
typeof NaN === "number". Not a Number is typed as a number. Also, NaN !== NaN — NaN is the only value not equal to itself.
